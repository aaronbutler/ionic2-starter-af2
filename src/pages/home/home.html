Ionic AngularFire2

This is intended as a starter template for a basic Ionic2/AngularFire2 application.

It does the following: Loads the required node modules Populates app.module.ts with stub firebaseConfig code Creates a data provider (FbdbService) with stub code to access a list and an object in the firebase database Publishes the FbdbService as a provider in app.module.ts Loads the FbdbService provider in the HomePage And creates this here page you are reading as we speak. It does not do the following: Teach you how to work with Ionic, Angular, Typescript, or Firebase. There are lots of resources for each of those Make your Firebase database for you Configure your Firebase security rules for you Provide an Authentication provider or an authentication object for you Promise to use best practices, just the practices that I tend to use myself Predict the future Guarantee that anything will work, initially or intermittently or permanently, if/when you try to use this starter template

If you immediately get errors about not finding AngularFire or related libraries, run ionic infoIf your Ionic CLI version is less than 2.1.18, you need to upgrade, at least for this to work. If you get lost, the docs will be your guide.
